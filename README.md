Rock, Paper, Scissors in JavaScript
========

Technologies used (so far)
----
- JavaScript
- Jasmine
- Travis CI

How to clone this repo
----
```sh
$ git clone git@github.com:kevinlanzon/rockPaperScissors-js.git
```

How to run tests
----
```sh
$ cd rockPaperScissors-js
$ open SpecRunner.html
```
